# StudentHub - Мобильное приложение для студентов

Полнофункциональное Flutter приложение для управления расписанием, задачами, оценками и коммуникацией студентов.

## 🚀 Быстрый старт

### Требования
- **Flutter**: 3.11.1+
- **Dart**: 3.11.1+
- **Android SDK**: 21+
- **Android Studio**: Latest version

### Установка

1. **Клонируйте репозиторий**
```bash
git clone <repository-url>
cd student_hub
```

2. **Установите зависимости**
```bash
flutter pub get
```

3. **Запустите приложение**

#### Для Android эмулятора:
```bash
# Запустите эмулятор
flutter emulators --launch Pixel_5

# Дождитесь загрузки и запустите приложение
flutter run
```

#### На реальном Android устройстве:
```bash
flutter run -d <device-id>
```

## 🔄 Backend Integration (Ready)

### Структура для API

Приложение подготовлено к интеграции с backend API. Текущие Mock репозитории легко заменяются на API:

#### Рекомендуемые endpoints для backend:

```
/auth
  POST /login       - Вход
  POST /register    - Регистрация
  
/schedule
  GET  /            - Получить расписание
  POST /            - Создать
  PUT  /{id}        - Обновить
  
/tasks
  GET  /            - Все задачи
  POST /            - Создать
  PUT  /{id}        - Обновить
  
/materials
  GET  /            - Все материалы
  GET  /{id}        - Детали
  
/grades
  GET  /            - Все оценки
  
/chats
  GET  /            - Чаты
  POST /{id}/messages - Сообщение
  
/notifications
  GET  /            - Уведомления
```
